# ORCA — Open Retail Command Agent

Autonomous inventory management AI agent system built with LangGraph, CrewAI, RAG, and FastAPI.

## Background
Rebuilt from a production system I deployed on Palantir AIP Studio for a UAE retail client (200 stores, 100+ SKUs).

## Stack
Python | LangGraph | CrewAI | ChromaDB | FastAPI | Streamlit | Docker | GCP
