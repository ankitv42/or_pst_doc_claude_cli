Fix 3 — docs/rag/retriever.py (is_available method, ~line 560)
    python
    # BEFORE
    def is_available(self) -> bool:
    return self._collection is not None

    # AFTER
    def is_available(self) -> bool:
        # Self-healing: if init failed earlier (e.g. model not ready during
        # FastAPI import chain), retry once now that everything is loaded.
        if self._collection is None:
            self._init()
        return self._collection is not None
    Why: Defensive safety net — if init ever fails at startup, retries on first check. Keep it, but Fix 1+2 were the real cure.
